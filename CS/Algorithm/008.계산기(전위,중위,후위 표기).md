# 계산기(중위, 후위 표기)

기본적으로 사용되는 표기식은 중위 표기식이다. (중위 표기 : 피연산자 사이에 연산자가 있는 표기법)

**중위 표기법을 후위 표기법으로 변환하는 알고리즘**

스택을 사용하여 중위 표기법을 후위 표기법으로 바꿀 수 있다.
