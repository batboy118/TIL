# 29. 로그인 없이 로그인하기 (ssh)

### ssh-keygen

```
ssh-keygen
cd ~
cd .ssh
ls -l
#id_rsa : 프라이빗 키 (절대 공유하면 안됨)
#id_rsa.pub : 퍼블릭키

```



내 컴퓨터에서 접근하려는 PC의 authorized_key 파일에 퍼블릭키를 덧붙여줌

```
cd ~/.ssh
ls -l
# authorized_key

```



내 컴퓨터에서 아래처럼 쓰면 대상 컴퓨터의 authorized_key에 저장이 된다.

```
ssh-copy-id egoing@192.222.222.222
```

아래를 치면 로그인 없이 접속이 가능하다.

```
ssh egoing@192.222.222.222
```
